BEM

-BEM là gì?

BEM là tiêu chuẩn đặt tên class khi viết CSS

-Ý nghĩa của BEM:

Là viết tắt của Block Element Modifier

Trong đó:
    Block: khối (container/cha)
    Element: thành phần trong khối (thành phần con)
    Modifier: bổ sung ý nghĩa cho Block/Element

-Ưu điểm:

Tính rõ ràng: do quy định đặt tên class cụ thể nên nhìn vô có thể nhận biết rõ ràng class dùng làm gì

=> Giúp vả team làm việc với nhau dễ dàng

Tái sử dụng dễ dàng: không sợ trùng tên class khi sử dụng nhiều chỗ trong project

Tính module: không lo CSS của class này ảnh hưởng lên CSS của class khác

-Nhược điểm:

Tên class dài, phức tạp

-Cú pháp:

.block

.block__element(2 dấu _ liên tiếp ngăn cách block và element)

.block--modifier(2 dấu - liên tiếp ngắn cách block/element với modifier)

block__element--modifier
